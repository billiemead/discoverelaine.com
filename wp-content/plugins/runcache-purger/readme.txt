Purge NGINX fastcgi, Proxy Cache and Redis Object Cache on RunCloud.io

== Description ==

This plugin will purge Runcloud.io NGINX fastcgi, Proxy Cache and Redis Object Cache.

== Installation ==

1. Install the plugin through your dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate Settings -> 'RunCache Purger' for configuration.
